# LG Task Hub - Prototipo web

Este prototipo funciona como una pagina web para coordinar el trabajo diario de la cuenta LG.

## Que incluye

- Tablero de avance general del dia.
- Checklist de tareas por responsable.
- Filtros por estado y persona.
- Anuncios generales e individuales.
- Solicitudes entre coordinador y entrenadores.
- Informacion interactiva de productos LG.
- Practica rapida de argumentos de venta.
- Guardado local en el navegador para probar el flujo.

## Como abrirlo

Abre el archivo `index.html` en un navegador moderno.

## Registro de entrenadores

1. El entrenador abre la web.
2. Entra a "Registro de entrenador".
3. Completa nombre, correo, zona o tienda y clave.
4. Presiona "Solicitar registro".
5. El coordinador entra como administrador.
6. Va a "Equipo".
7. En "Alta de entrenadores", aprueba la solicitud.
8. El entrenador ya puede ingresar con su correo y clave.

## Como convertirlo en pagina web real

Para usarlo con todo el equipo, el siguiente paso seria conectarlo a:

- Inicio de sesion para coordinador y entrenadores.
- Base de datos para guardar tareas, anuncios y avances.
- Panel de administracion para crear usuarios.
- Notificaciones por correo, WhatsApp o push.
- Hosting web para acceder desde celular o computadora.

## Hosting gratis recomendado

Opcion rapida: GitHub Pages.

1. Crear una cuenta en GitHub.
2. Crear un repositorio publico llamado `TUUSUARIO.github.io`.
3. Subir estos archivos al repositorio: `index.html`, `styles.css`, `app.js` y `README.md`.
4. Entrar a Settings > Pages.
5. Seleccionar "Deploy from a branch".
6. Elegir la rama principal y guardar.
7. Abrir `https://TUUSUARIO.github.io/`.

Con el usuario detectado `ANGELHOWARD12`, el nombre seria:

- Repositorio: `ANGELHOWARD12.github.io`
- Web: `https://ANGELHOWARD12.github.io/`

Nota: un dominio propio como `.com` normalmente no es gratis. Lo gratuito seria el subdominio `github.io`.

## Roles sugeridos

- Coordinador: crea tareas, publica anuncios, revisa avance y responde solicitudes.
- Entrenador: revisa tareas asignadas, marca avances, envia solicitudes y consulta informacion LG.
